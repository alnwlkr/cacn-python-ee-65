หนูเจอรี่ในมัลติเวิร์สมีอยู่หลายตัวด้วยกัน
กำหนดให้มีหนูเจอรี่อยู่ N ตัว มีรูหนูอยู่ N รู
รูหนูทุกรู และหนูทุกตัวอยู่บนแนวเส้นตรงเดียวกัน (ในแนวแกน X)
หนูแต่ละตัวเคลื่อนที่ด้วยความเร็ว 1 เมตร ต่อ วินาที


เมื่อทอม



โผล่เข้ามาบริเวณบ้านของเจอรี่ เจอรี่ทุกตัวจะวิ่งเข้าไปในรูพร้อม ๆ กัน
จงหาเวลาที่เร็วที่สุด ที่เป็นไปได้ที่เจอรี่ตัวสุดท้าย จะวิ่งเข้าไปในรู


* หนูเจอรี่จะวิ่งเข้าบ้านหลังไหนก็ได้ แต่จะเข้าบ้านละ 1 ตัวเท่านั้น


 

ตัวอย่างวิธีคิด


สมมุติว่ามีเจอรี่อยู่ 3 ตัว และรูหนู 3 รู โดยที่เจอรี่ทั้ง 3 ตัว อยู่ในแนวเส้นตรง (แกน x) ที่ 4, -4, 2 ตามลำดับ (หน่วยเป็นเมตร)
และรูหนู 3 รู อยู่ที่ตำแหน่งในแนวเส้นตรง (แกน x) ที่ 4, 0, 5 (หน่วยเป็นเมตร)

ตัวอย่างของคำตอบที่เป็นไปได้เป็นดังนี้
เจอรี่ตัวที่ 1 ที่อยู่ตำแหน่งที่ x = 4 ควรวิ่งเข้าไปในบ้านที่ x = 4 และใช้เวลาเท่ากับ 0 วินาที
เจอรี่ตัวที่ 2 ที่อยู่ตำแหน่งที่ x = -4 ควรวิ่งเข้าไปในบ้านที่ x = 0 และใช้เวลาเท่ากับ 4 วินาที
เจอรี่ตัวที่ 3 ที่อยู่ตำแหน่งที่ x = 2 ควรวิ่งเข้าไปในบ้านที่ x = 5 และใช้เวลาเท่ากับ 3 วินาที
แสดงว่าเจอรี่ตัวสุดท้าย (ในตัวอย่างคือเจอรี่ตัวที่ 2) จะวิ่งเข้าไปในบ้านให้เร็วที่สุด จะใช้เวลา 4 วินาที

ไม่มีวิธีอื่นที่จะทำให้หนูตัวสุดท้ายวิ่งเข้าบ้านได้เร็วกว่านี้แล้ว


 

Specification
INPUT SPECIFICATION	
บรรทัดที่ 1: N เป็นจำนวนหนูและจำนวนบ้าน (จำนวนนับ)
บรรทัดที่ 2 ถึงบรรทัดที่ N+1: เป็นตำแหน่งของหนูแต่ละตัวในแนวแกน x (หน่วยเป็นเมตร) เป็นจำนวนเต็ม
บรรทัดที่ N+2 ถึงบรรทัดที่ 2N+1: เป็นตำแหน่งของบ้านแต่ละหลังในแนวแกน x (หน่วยเป็นเมตร) เป็นจำนวนเต็ม
OUTPUT SPECIFICATION
เป็นเวลาที่น้อยที่สุดที่หนูตัวสุดท้ายวิ่งเข้าบ้าน (วินาที)
