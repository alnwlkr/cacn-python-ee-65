ข้อนี้อาจารย์ไม่ได้ลงโจทย์ไว้ในระบบแล้วผมก็ไม่ได้เซฟไว้เสียด้วย
โจทย์คือ ใส่ชื่อยูทูปเบอร์ และจำนวนซับลงไป จนเจอคำว่า ~END~
ผลลัพธ์ คือ ให้แสดงผลชื่อยูทูปเบอร์ตามด้วยซับ โดยให้เรียงตามซับ 
หากซับเท่ากัน ให้เรียงตามชื่ออีกที ถ้าชื่อตรงกันอีกให้เรียงตามตัวอักษร

INPUT
HRK 10M
Bearhug 900K
Bearhouse 900K
9arm 632K
RUBSARB production 1.32M
Bayriffer 4.14M
~END~

OUTPUT
HRK 10000000
Bayriffer 4140000
RUBSARB production 1320000
Bearhouse 900000
Bearhug 900000
9arm 632000

ข้อนี้ขอแค่ไกด์ไว้ละกันนะครับเนื่องจากโจทย์อาจจะไม่ค่อยชัดเจน
- รับค่ามา แล้วแตกเป็น list 2 มิติ 
ex [['HRK', '10M'], ['Bearhug', '900K']]
- เปลี่ยนค่าที่ 2 ของ list เป็นตัวเลข M --> 1_000_000 K --> 1_000
ex [['HRK', 10000000 ], ['Bearhug', 900000]]
- sort โดยใช้ key เป็นข้อมูลลิสดัชนีที่ 1 (ซับ) และ ข้อมูลลิสดัชนีที่ 0 (ชื่อช่อง).lower() ด้วย
ตรงนี้ตอนสอบผมทำไม่เป็น เลยไม่ได้ใช้ sort เรียงซับแต่เอามาใช้เรียงชื่อแทน
ผมลองไปหาข้อมูลมา วิธีทำ sort list 2 มิติ เหมือนจะต้องใช้ lambda แต่เหมือนตอนนั้นผมจะใช้ * ในการช่วย sort
